# AI Subtitle Generator

## Problem
Manually creating subtitles for video takes hours in post production.

## Solution
AI-powered subtitle generator using OpenAI Whisper.
Upload any video → get a proper .srt subtitle file in seconds.

## What it does
- Accepts video or audio files (mp4, mov, mp3, wav)
- Transcribes speech using OpenAI Whisper AI
- Generates industry-standard .srt subtitle file with timestamps
- Works in 99 languages automatically

## Tech Stack
Python | OpenAI Whisper | Streamlit | ffmpeg

## How to Run
pip install openai-whisper streamlit
streamlit run app.py

## Use Cases
- Netflix / YouTube subtitle generation
- Post production workflows
- VFX studio deliverables
- Content localization
